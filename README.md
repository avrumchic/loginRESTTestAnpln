# RestClient Automation Project

This is a sample automation project for rest client tests
using OkHttp3 client
Unit Testing framework is TestNg

### Prerequisites ###
maven
Java 8

### To Build the project
>mvn clean install

### To run the tests
>mvn clean test

## Test Results should be shown in console:
